# 🚀 YOLO Explorer: AI-Powered Real-Time Object Detection
## 🔍 Overview
YOLO Explorer is an advanced object detection system leveraging YOLOv8 to identify objects in real-time across various environments. Combining computer vision with a full-stack web interface, this project demonstrates practical applications of AI in open-world scenarios.

📌 **Key Applications:**
- Smart surveillance systems
- Autonomous vehicle navigation
- Industrial quality control
- Augmented reality integration

## 🚀 Quick Start

### Prerequisites
- Node.js v16+
- Python 3.8+
- MongoDB Atlas account

### Installation
```bash
# Clone the repository
git clone https://github.com/Deb2oo/YOLO-Explorer.git

# Install dependencies
cd YOLO-Explorer
npm install
pip install -r requirements.txt

# Set up environment variables
cp .env.example .env

# Run the application
npm run dev
```
